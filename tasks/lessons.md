# Lessons

## 2026-02-24

- Ошибка: при удалении плавающей кнопки Framer был применен слишком широкий regex по `<script>`, что удалило критичные блоки и вызвало белый экран.
- Правило: никогда не удалять блоки в экспортированном HTML широкими шаблонами вида `.*__token.*</script>`; использовать только точечное удаление по точному `id`/фрагменту.
- Проверка перед push: обязательно убедиться, что в `index.html` присутствуют `id="main"` и `data-framer-bundle="main"`.
